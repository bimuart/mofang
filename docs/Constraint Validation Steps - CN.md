3x3魔方约束校验
├── 棱块位置
│   ├── 取填了 2 格的棱块，若其颜色组合不存在于棱块的 12 个棱颜色组合 -> 不通过
│   ├── 若所填棱块数超过棱块的 12 个棱颜色组合中可满足的数 -> 不通过
│   └── 其他情况 -> 通过
├── 角块位置
│   ├── 取填了 2 或 3 格的棱块，若其颜色组合不存在于角块的 8 个角颜色组合 -> 不通过
│   ├── 所填角块数超过角块的 8 个角颜色组合中可满足的数 -> 不通过
│   └── 其他情况 -> 通过
├── 棱块翻转
│   ├── 若存在某棱槽 2 格都未填 -> 通过
│   ├── 若 12 个棱槽 2 格均为有效面色（棱完全填充） -> 计算校验 -> 不通过 / 通过
│   └── 若已填 12 个棱槽但存在未填的格 -> 枚举所有情况
│       ├── 所有枚举中若存在通过的情况 -> 通过
│       └── 否则 -> 不通过
├── 角块扭转
│   ├── 若存在某角槽三格都未填 -> 通过
│   ├── 若 8 个角槽 3 格均为有效面色（角完全填充） -> 计算校验 -> 不通过 / 通过
│   └── 若已填 8 个角槽但存在未填的面 -> 枚举所有情况
│       ├── 所有枚举中若存在通过的情况 -> 通过
│       └── 否则 -> 不通过
└── 棱角奇偶
    ├── 若已填的棱块 / 角块可行排列的数量 > 3 种 -> 通过
    ├── 若棱块 / 角块有 2 个以上槽位未填 -> 通过
    └── 否则 -> 枚举所有可能，并用棱块翻转 / 角块扭转校验过滤
        ├── 所有枚举中若存在通过的情况 -> 通过
        └── 否则 -> 不通过
